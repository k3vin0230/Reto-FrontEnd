
# Instalación y uso de repositorio FrontEnd

### 1. Clonar Repositorio a máquina local

```bash
git clone <repository-url>

```

### 2. Instalar Dependencias

```bash
npm install

```

### 3. Ejecutar El proyecto

```bash
ng serve

```
